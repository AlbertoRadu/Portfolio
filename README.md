# Alberto Radu - Developer Portfolio

Mi portafolio web personal donde muestro mis proyectos, competencias técnicas e historial profesional.

## Tecnologías Principales

- **Frontend:** React (TypeScript) + Vite
- **Estilos:** Tailwind CSS (Diseño adaptable, modo oscuro nativo, barra de desplazamiento personalizada)
- **Animaciones:** GSAP + ScrollTrigger (Efectos de desplazamiento interactivo y textos dinámicos)
- **Modelo 3D:** Integración con Google `<model-viewer>` (Auto-rotación interactiva desacelerada por scroll)
- **Internacionalización:** Contexto multilingüe personalizado (Español, Catalán e Inglés)
- **CI/CD:** Despliegue automatizado a GitHub Pages mediante GitHub Actions

## Desarrollo Local

Para ejecutar el proyecto en tu máquina local:

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm run dev

# Compilar para producción
pnpm run build
```
